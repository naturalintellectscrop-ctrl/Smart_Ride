# Smart Ride Mobile Assets

This folder contains app assets for the Smart Ride mobile application.

## Required Assets

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image (1284x2778)
- `adaptive-icon.png` - Android adaptive icon foreground (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Asset Guidelines

- Use high-resolution images for better quality
- Keep file sizes optimized
- Follow platform-specific guidelines for icons
